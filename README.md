# Typescript React boilerplate 20180511

## Stack

* typescript
* react
* styled-components
* netlify
* jest
* prettier
* tslint
* storybook

## Commands

* `yarn watch`: strat dev server(localhost:8080)
* `yarn storybook`: start storybook(localhost:6006)
* `yarn test`: run jest
* `yarn lint`: run tslint
* `yarn deploy`: deploy to netlify
